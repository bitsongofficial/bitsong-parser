# BitSong Blockchain Parser

BitSong Blockchain Parser in Node.js + MongoDB

## Features

- Parsing entire blockchain
- Retrieving transactions

## Install Locally

- Install required modules:
  `$ npm install`
- Compile TypeScript:
  `$ npm run build`
- Start the app:
  `$ node dist/server.js`
